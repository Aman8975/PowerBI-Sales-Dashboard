# Power BI Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard created to analyse sales performance, profit trends and discount patterns across different cities and time periods.

## Key Insights
- Total Orders analysis
- Profit vs Net Sales comparison
- Average Discount by Promotion Category
- Sales trends over time
- City-wise sales distribution

## Tools Used
- Power BI
- Data Visualization
- Business Intelligence techniques

## File Included
- Power BI Dashboard (.pbix)

## Purpose
The aim of this project is to demonstrate data analysis and visualization skills using Power BI to generate meaningful business insights.
